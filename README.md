# Proyecto Página Web

Este proyecto contiene una estructura básica profesional para el desarrollo de sitios web estáticos.

## Estructura de Archivos

- `index.html`: Punto de entrada principal con estructura semántica HTML5.
- `css/styles.css`: Estilos organizados con un reset básico y tipografías modernas.
- `js/main.js`: Lógica de JavaScript encapsulada para evitar polución del scope global.

## Cómo empezar

1. Abre el archivo `index.html` en tu navegador.
2. Edita `css/styles.css` para cambiar la apariencia.
3. Añade interactividad en `js/main.js`.
